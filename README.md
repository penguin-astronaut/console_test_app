# Weather App

Тестовое задание

### Установка зависимостей
    composer install

### Запуск приложения
    php app.php

### Конфигурация config.json
    "folder": "src" - путь от начала проекта, в котором будет производиться поиск
    "ext": ["js", "php", "html"] - расширения файлов, которые будут преобразованы в pdf
